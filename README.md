## 課程簡介

由於 Vibe Coding 日新月異，模型每天都有變化，因此本課程會動態調整模型，並且提供相關的程式碼範例。但大方向如下：

1. 大型語言模型的基礎知識
2. Vibe Coding 的基礎知識和使用
3. 利用現有的網頁工具進行 Vibe Coding
4. Cursor 的基礎/進階使用
5. Cursor 開發各種專案
6. Claude Code 的基礎/進階使用
7. Claude Code 開發各種專案
